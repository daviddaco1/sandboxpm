# @sandboxpm/config

Loads and merges `.sandboxpmrc` config files for [sandboxpm](https://www.npmjs.com/package/sandboxpm), a security-first, zero-trust package manager for Node.js.

Internal building block of sandboxpm — not intended for standalone use. See the [main repository](https://github.com/daviddaco1/sandboxpm) for documentation.
